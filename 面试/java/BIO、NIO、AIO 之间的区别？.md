
| 特性    | BIO          | NIO            | AIO            |
| ----- | ------------ | -------------- | -------------- |
| 工作模式  | 同步阻塞         | 同步非阻塞          | 异步非阻塞          |
| 线程需求  | 高，每个IO连接一个线程 | 低，一个线程管理多个IO连接 | 更低，依赖操作系统通知    |
| 缓冲区使用 | 基本不使用        | 广泛使用           | 广泛使用           |
| 连接数   | 小            | 大              | 非常大            |
| 场景    | 传统Socket编程   | 网络聊天室、即时通讯等    | 高性能网络服务器、群聊服务器 |
