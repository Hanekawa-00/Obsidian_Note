
| 特性       | `CountDownLatch`                   | `CyclicBarrier`                  |
| -------- | ---------------------------------- | -------------------------------- |
| **设计目的** | 等待多个线程完成任务                         | 一组线程相互等待，直到所有线程到达屏障点             |
| **使用方式** | 基于计数器，调用 `await()` 和 `countDown()` | 调用 `await()`，等待所有线程到达屏障点         |
| **重用性**  | 一次性使用，不能重置                         | 可重复使用，屏障触发后自动重置                  |
| **适用场景** | 一个或多个线程等待其他线程完成任务                  | 一组线程相互等待，分阶段执行任务                 |
| **屏障动作** | 不支持
